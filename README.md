# 🐝 SKBossBee

**SKBossBee** — это легкий и функциональный плагин для Minecraft (Spigot/Paper), который привносит на сервер динамическое событие с боссами-пчелами. Плагин автоматически строит арену по расписанию, спавнит усиленных мобов и выдает награды победителям.

---

## ✨ Основные возможности

* **📅 Автоматизация:** Запуск ивента строго по расписанию, указанному в конфигурации.
* **🏗️ Интеграция со схематиками:** Автоматическая вставка арены (через WorldEdit) при старте и полная очистка территории после завершения.
* **🛡️ Защита региона:** Автоматическое создание региона WorldGuard на время ивента для предотвращения гриферства.
* **🐝 Настраиваемые боссы:** Полный контроль над здоровьем, уроном, именами и количеством пчел.
* **🧪 Боевые эффекты:** Наложение негативных эффектов (яд, медлительность и др.) на игроков при атаке босса.
* **💰 Система наград:** Выполнение консольных команд (выдача предметов, валюты) для игрока, нанесшего последний удар.
* **📍 Гибкий спавн:** Поддержка как статичной точки появления, так и случайного спавна по миру в заданном радиусе.

---

## 💻 Команды и Права

| Команда | Описание | Право (Permission) |
| :--- | :--- | :--- |
| `/bossbea start` | Принудительно запустить ивент | `bossbea.admin` |
| `/bossbea stop` | Принудительно завершить (убить пчел и убрать арену) | `bossbea.admin` |
| `/bossbea setspawn` | Установить текущую точку как место ивента | `bossbea.admin` |
| `/bossbea reload` | Перезагрузить конфигурацию плагина | `bossbea.admin` |
| `/bossbea delay` | Узнать время до следующего ивента | `bossbea.delay` |

---

## 🛠 Требования

Для корректной работы плагина необходимы:
1.  **WorldEdit** (или FastAsyncWorldEdit) — для работы со схематиками.
2.  **WorldGuard** — для защиты территории ивента.
3.  **Java 16+** (рекомендуется 17).

## ⚙️ Пример конфигурации (config.yml)

```yaml
# Список времени запуска (HH:mm)
event-times:
  - "12:00"
  - "18:00"

# Команды при смерти каждой пчелы
rewards:
  - "eco give %player% 500"
  - "give %player% diamond 5"

bee:
  amount: 5          # Количество пчел
  health: 150.0      # ХП пчелы
  damage: 8.0        # Урон (1.0 = полсердечка)
  name: "&6Пчела-Босс"
  effects:
    POISON:
      level: 1
      duration: 100

spawn:
  world: "world"
  static: true       # true - в одной точке, false - рандом
  x: 0.0
  y: 70.0
  z: 0.0
  random-radius: 1000

messages:
  prefix: "&8[&6BossBee&8]"
  start: "&fИвент начался на координатах &e%loc%&f!"
  stop: "&fВсе боссы повержены, ивент окончен."
  reward: "&aВы получили награду за убийство босса!"
  delay: "&fДо следующего ивента: &e%h% ч. %m% мин."
